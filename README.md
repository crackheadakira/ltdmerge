# LTD Merge

ltdmerge is a tool to streamline the process of creating new items in the Mii editor.

# Usage

## Creating a mod

```sh
ltdmerge add --base /path/to/game/romfs --model /path/to/your/custom/model --icon /path/to/your/png --out /path/to/output_dir/
```

## Merging mods

```sh
ltdmerge merge path/to/mod1 path/to/mod/1 --out /path/to/output_dir/
```
